Change username, password and database path in src/DBConnect to represent your database

Open terminal in root folder

For windows
Compile:
javac -cp "lib/mysql-connector-j-9.4.0.jar" src\*.java -d out
Run:
java -cp "out;lib/mysql-connector-j-9.4.0.jar" Main

For Mac
Compile:
javac -cp "lib/mysql-connector-j-9.4.0.jar" src/*.java -d out
Run:
java -cp "out:lib/mysql-connector-j-8.4.0.jar" Main
